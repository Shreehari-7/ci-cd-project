# CI/CD Pipeline Project

## 📌 Description
This project demonstrates the implementation of a CI/CD pipeline for a web application using GitHub Actions.

## 🚀 Features
- Automated build using GitHub Actions
- Continuous Integration on every push
- Continuous Deployment using GitHub Pages
- Simple web interface

## 🛠️ Technologies Used
- Node.js
- GitHub Actions
- GitHub Pages

## 🔄 Workflow
1. Code pushed to GitHub
2. CI pipeline runs automatically
3. Application is deployed automatically
